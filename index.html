Languages

  • Python

    Python was the first programming language that I learnt. In 2020 I completed the Python for Everybody specialization on Coursera taught by Charles Severance. In 2022, in the first year of my degree, Python was the language used in the first year unit Programming Principles. I am familiar with the fundamentals of Python, including data types, flow control, functions, classes, and data structures. Some of the libraries that I have used are BeautifulSoup4, Tkinter, discord.py, and requests.

  • C++

    C++ was my first introduction to a C style language and my first statically typed, compiled language. I initially learnt C++ by converting one of my Python assignments from my first year unit (Programming Principles) into C++. In my second year we learnt C++ in the unit Object-Oriented Programming with C++. I prefer C++ over Python due to the fact that it is statically typed and the way that it uses header files for implementing classes. I am familiar with C++ data types, flow control, functions, classes and structs, inheritance, pointers and references, and overloaded functions/operators.
