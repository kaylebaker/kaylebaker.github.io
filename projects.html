Projects

Here are some projects I have worked on:

  • Hugin-bot: a Discord bot for Valheim

    A bot I built for my Valheim Discord server. Hugin-bot can return the latest Valheim news, details on bosses and enemies, recipes, and food effects. You can find the README at https://github.com/kaylebaker/hugin-bot

    Skills developed:

    • Using APIs and API keys
    • Scraping data using BeautifulSoup4
    • discord.py commands and asynchronous functions

    What I learned:

    • How to store environmental variables using dotenv
    • Connecting a bot to a Discord server
    • Using decorators for Python functions

  • Text Adventure game in Godot

    Developed a satrical text adventure style game centered around the Com Sci building at ECU. The project was developed in Godot, with the help of online tutorials, and modified to suit my needs and requirements.

    Skills involved:

    • Coding in GDScript
    • Understanding of object-oriented design
    • Utilising the different components of a game engine

    What I learned:

    • Using the node structure of the Godot game engine and creating components that work together
    • The use of the "separation of concerns" methodology in developing a project with various scripts and components
    • Game development is complicated and difficult! Seriously, hats off to indie game developers out there
